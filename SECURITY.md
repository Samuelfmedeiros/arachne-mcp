# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

**DO NOT** create public GitHub issues for security vulnerabilities.

Contact Samuel Medeiros directly:
- **Email:** samuelandrademedeiros@gmail.com
- **GitHub Security Advisory:** https://github.com/Samuelfmedeiros/arachne-mcp/security/advisories/new

We will acknowledge receipt within 48 hours and provide a timeline for the fix.

## Security Practices

1. **No secrets in code** — All API keys, tokens, and passwords are loaded from environment variables.
2. **Secret Scanning** — GitHub Secret Scanning + Push Protection enabled.
